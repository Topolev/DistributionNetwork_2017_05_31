Создать/редактировать характеристику

Пока не задано ни одной точки характеристики. Для добавления новой точки воспользуйтесь соответствующей кнопкой выше. Для характеристики необходимо задать как минимум 2 точки.
Обратите внимание, что для корректного тображения характеристики необходимо задавать точки в порядке возрастания тока, начиная с точки с наименьшим значением тока
Ток, A Время, с
