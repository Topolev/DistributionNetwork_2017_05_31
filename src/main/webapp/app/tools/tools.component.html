Построение время-токовых характеристик

Для оценки селективности срабатывания защит часто приходится строить время-токовые характеристики. Для их построения можно использовать различные подручные средства: начиная с построения в ручную на листе бумаге, заканчивая использованием MathCad. Excel. Мы разаработали свой инструмент, который в себя будет включать только самое необходимое, что надеемся в конечном итоге сможет сохранить Вам время и сфокусироваться на более важных расчетах.

РЗА, Расчет уставок, Автоматические выключатели, Карта селективности
